About Arch Linux

Arch Linux is an independently developed, x86-64 general purpose GNU/Linux distribution versatile enough to suit any role. Development focuses on simplicity, minimalism, and code elegance. Arch is installed as a minimal base system, configured by the user upon which their own ideal environment is assembled by installing only what is required or desired for their unique purposes.

Strong community

Our community is diverse and helpful, and we pride ourselves on the range of skillsets and uses for Arch that stem from it.

Wiki

Our wiki is the place to learn about Arch. Look for guides on the forum whenever you need help with your system, from installation to everyday use!

KISS

Here we follow the KISS style! our system was designed to be simple and only work with what you need, making you the true owner of the system.

Rolling release

Arch Linux uses a "rolling release" system which allows one-time installation and perpetual software upgrades. It is not generally necessary to reinstall or upgrade your Arch Linux system from one "version" to the next.

Pacman

Arch Linux uses its own Pacman package manager. which couples simple binary packages with an easy-to-use package build system.

AUR

Arch also offers the Arch Linux User Repository (AUR), which contains more than 49,000 build scripts, for compiling installable packages.
